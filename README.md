# Suspended Ceiling Calculator

React + TypeScript приложение за изчисляване на окачени тавани по системи Knauf D11. Интерфейсът е на български и е насочен към бърза ориентировъчна калкулация на профили, окачвачи, връзки, дюбели, винтове и основна работна схема за една или повече стаи.

## Основни възможности

- Избор на конструкционна система: `D111`, `D112`, `D113`, `D116` или `Custom`.
- Поддръжка на множество стаи в един проект.
- Автоматично попълване на разстоянията `a`, `b` и `c` по въведените таблици на Knauf.
- Ръчна корекция на разстоянията с ясно отбелязване дали стойността е "по Knauf" или "ръчно".
- Избор на клас натоварване, огнезащита и тип/дебелина гипсокартон.
- Валидиране на комбинации и предупреждения за важни ограничения.
- SVG визуализация на носещи профили, монтажни профили, окачвачи и удължители.
- Обобщена таблица с количества по стаи.
- Общ материален списък с конфигурируем резерв.
- Локално запазване в браузъра чрез `localStorage`.
- Импорт и експорт на JSON.
- Експорт на активно изчисление с формули и позиции.
- Прост Excel XML експорт на количествата.

## Как се използва

1. Въведете име и размери на стаята в сантиметри.
2. Изберете конструкция, натоварване, огнезащита и тип гипсокартон.
3. Проверете автоматично зададените стойности за `a`, `b`, `c`, начално отстояние и UD дюбели.
4. Ако е нужно, коригирайте стойностите ръчно.
5. Прегледайте резултатните карти, схемата и таблицата с количества.
6. Добавете допълнителни стаи чрез "Нова стая".
7. Експортирайте проекта или резултатите чрез JSON/Excel бутоните.

## Поддържани системи

| Система | Описание |
| --- | --- |
| `D111` | Дървена подконструкция |
| `D112` | Метална подконструкция |
| `D113` | Метална подконструкция на едно ниво |
| `D116` | UA/CD система за големи отвори |
| `Custom` | Ръчна конструкция със свободни стойности |

## Речник на съкращенията

### Огнезащита и EI класове

| Съкращение | Значение |
| --- | --- |
| `EI` | Клас на пожароустойчивост. `E` означава цялост на конструкцията, а `I` означава топлоизолационна способност при пожар. |
| `EI30` | Конструкцията трябва да запази цялост и изолационна способност поне 30 минути при стандартно пожарно изпитване. |
| `EI60` | Същото изискване за минимум 60 минути. |
| `EI90` | Същото изискване за минимум 90 минути. |
| `EI120` | Същото изискване за минимум 120 минути. |
| `отдолу` | Огневото въздействие е откъм помещението, под окачения таван. |
| `отгоре` | Огневото въздействие е от кухината/междинното пространство над окачения таван. |

В приложението текущата опция `Огнезащита по D11 таблица` означава, че се използват fire таблиците от D11 документа, но конкретният EI клас и посоката на огневото въздействие още трябва да се проверят отделно спрямо системния детайл.

### Профили и елементи

| Съкращение | Значение |
| --- | --- |
| `CD 60/27` | Метален таванен профил с приблизителни размери 60 x 27 mm. Използва се за носещи и/или монтажни редове при D112/D113 и като CD част при D116. |
| `UD 28/27` | Периферен профил по стените. Служи за водене/оформяне на ръба на конструкцията. |
| `UA 50/40` | Усилен метален профил, използван при D116 за по-големи отвори/разстояния. |
| `UW 50/40` | U-образен профил, използван в някои D116 детайли, например около UA удължавания. |
| `W профил` | Специален вариант профил при част от D112 решенията. Не е моделиран в текущия `a/b/c` калкулатор, защото таблицата му е по различен параметър. |
| `окачвач` | Елемент, който свързва таванната конструкция към основния таван. Може да бъде директен, нониус, анкерфикс, акустичен и др. |
| `нониус` | Регулируем тип окачване, използван при по-големи височини или специфични системни решения. |
| `анкерфикс` | Вид окачващ/закрепващ елемент за CD профили според системния детайл. |

### Типове окачвачи

| Тип | Кога се избира |
| --- | --- |
| `Общ/неуточнен окачвач` | За `Custom` или когато системният детайл още не е избран. Не доказва носимоспособност по Knauf. |
| `Директен окачвач за CD` | За обичайни D112/D113 CD конструкции с малки до средни височини на окачване и без специално изискване за 0.40 kN. |
| `Нониус окачвач` | При по-високи товари, по-голяма височина на окачване или когато footnote изисква окачвач/закрепване 0.40 kN. |
| `Анкерфикс за CD` | За CD системи, когато системният детайл допуска/изисква закрепване клас 0.40 kN. |
| `Акустичен окачвач` | При акустични изисквания и нужда от ограничаване на вибрации. Носимоспособността трябва да се провери по конкретния продукт. |
| `UA/M8 окачване` | За D116 системи с UA профили и по-големи отвори. Проверява се по D116 детайла и избрания вариант. |

### Разстояния и товари

| Означение | Значение |
| --- | --- |
| `a` | Разстояние между окачвачите по носещия профил, в mm. |
| `b` | Разстояние между монтажните профили/летви, в mm. Зависи основно от типа и дебелината на плоскостта. |
| `c` | Разстояние между носещите профили/летви, в mm. |
| `kN/m2` | Килонютон на квадратен метър. Използва се за клас натоварване на таванната система. |
| `0.15`, `0.30`, `0.40`, `0.50`, `0.65` | Товарни класове в `kN/m2`, използвани в таблиците. |
| `0.40 kN` | Носимоспособност на окачвач/закрепване, отбелязвана във footnotes за по-високи товарни класове. |

### Плоскости и крепежи

| Съкращение | Значение |
| --- | --- |
| `A` | Стандартна гипсокартонена плоскост. |
| `H2` | Импрегнирана влагоустойчива гипсокартонена плоскост. |
| `DF/GKF` | Пожарозащитна гипсокартонена плоскост. |
| `Diamant` | Специална Knauf плоскост с повишени механични/акустични характеристики според системата. |
| `Silentboard` | Акустична плоскост с по-висока маса, използвана за звукоизолационни решения. |
| `TN` | Винтове за закрепване на гипсокартон към метална конструкция. |
| `LN` / `LB` | Винтове за метални връзки/профили според системния детайл. |
| `Trenn-Fix` | Разделителна лента по периферията, използвана за контролирано отделяне на шпакловката от съседни конструкции. |
| `фугопокривна лента` | Лента за армиране на фугите между гипсокартонените плоскости. |
| `шпакловка за фуги` | Материал за запълване и обработка на фугите и винтовете. |
| `минерална вата` | Изолация в кухината над тавана; използва се за акустика и/или при някои огнезащитни решения. |

### Материали за обшивка и довършване

Приложението изчислява ориентировъчни количества за:

- гипсокартонени плоскости по площ, размер на листа и брой слоеве;
- `TN` винтове за закрепване на плоскостите;
- `LN/LB` винтове за метални връзки;
- фугопокривна лента;
- шпакловка за фуги;
- `Trenn-Fix` по периметъра;
- минерална вата, когато е включена от настройките или е избрана огнезащита.

Тези количества са estimate. Реалната разкройка на плоскостите, отворите, фугите, технологичните застъпвания и конкретната степен на шпакловане могат да променят покупния списък.

При стандартните Knauf типове броят слоеве се взима от избрания тип плоскост, например `2x12.5` означава два слоя. Глобалната настройка `Custom слоеве плоскости` се използва само когато типът плоскост е `Custom / ръчно b`.

## Важни ограничения

Приложението е помощен калкулатор и не замества проектантско решение, техническа спецификация или официален системен разход от Knauf.

Текущият материален списък е геометричен summary. При някои системи все още не се разделят всички реални системни компоненти. Например:

- `D111` не отделя детайлно дървените летви по варианти.
- `D116` все още не разделя UA, CD и UW компоненти в отделни редове.
- Връзки, окачвачи и удължители са изчислени в опростен вид.
- Някои footnotes и системни условия от документацията са представени като предупреждения, но не всички са моделирани напълно.

Преди реална поръчка или монтаж проверете резултатите спрямо актуалната техническа документация и конкретните условия на обекта.
